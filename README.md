# Weather Planner

## Purpose
This application allows people to see the weather for various cities. People will use this to plan trips on sunny days. When a city is searched, the present and future weather will be shown.

## Built with
* HTML
* CSS
* JS
* Bootstrap
* MomentJS
* localStorage
* Server Side API
